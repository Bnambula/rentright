# 🏠 RentRight! — Uganda's Verified Rental Marketplace

> Safe, verified, affordable homes in Uganda. No brokers. No scams. GPS-confirmed listings.

## 🌍 Live Demo
[rentright.vercel.app](https://rentright.vercel.app)

## ✨ Features

- 🔍 **Smart Search** — Filter by location, budget, lifestyle & atmosphere mode
- 🛡️ **6-Layer Verification** — NIN, GPS lock, AI selfie match, photo geo-tagging, device fingerprint
- 📅 **Booking System** — Pay viewing fee via MTN MoMo, digital agreement, contact unlock
- 🔒 **Contact Protection** — Landlord details hidden until viewing fee paid
- 🤖 **AI Housing Assistant** — Claude-powered chat for personalised recommendations
- 🌿 **Living Environment Score** — Safety, cleanliness & accessibility rated per listing
- 🔁 **Referral System** — Earn UGX 20,000 per successful referral via MoMo
- 🚀 **Boosted Listings** — Landlord monetisation (Free / Boost / Premium / Verify)
- 🗺️ **Comfort Zone Map** — Visual area heatmap (GPS integration in Phase 3)
- 📱 **Mobile-First** — Designed for Uganda mobile usage patterns

## 💰 Revenue Model

| Stream | Who Pays | Amount |
|--------|----------|--------|
| Viewing fee | Tenant | UGX 2,000–10,000 |
| Boost listing | Landlord | UGX 25,000/week |
| Premium listing | Landlord | UGX 50,000/month |
| Verification service | Landlord | UGX 20,000–80,000 |
| Commission | Landlord | 3–7% first month rent |

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## 🌐 Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import project
3. Select this repo → Deploy (auto-detected as Vite)
